راهنما (آفلاین واقعی)
1) اگر می‌خواهی بدون هیچ دردسری و حتی بدون Extract اجرا شود:
   فایل MindIslands_Single.html را باز کن. (همه چیز داخل همین یک فایل است)

2) اگر می‌خواهی PWA / Service Worker داشته باشی (کش آفلاین، نصب روی مرورگر):
   پوشه folder/ را Extract کن و با یک سرور محلی اجرا کن:
     python -m http.server 8080
   سپس برو روی:
     http://localhost:8080/folder/

نکته: باز کردن فایل‌ها از داخل ZIP (بدون Extract) معمولاً باعث می‌شود JS/CSS لود نشوند و صفحه خالی شود.
